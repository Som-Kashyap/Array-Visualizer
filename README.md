# Array Visualizer v_1.1

An interactive web application built using HTML,CSS,and JavaScript to visualize common array operations through a simple user interface.

## Features

- Add (Push) elements into the array
- Remove (Pop) the last element
- Display array contents visually
- Display array length
- Search for an element and view its index
- Display first element
- Display last element
- Display sum
- Display operations description
